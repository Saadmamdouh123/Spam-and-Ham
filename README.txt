# 🔍 Filtrage de Données avec Pandas - Filter2.ipynb

Ce projet présente un notebook Jupyter (`Filter2.ipynb`) conçu pour effectuer des opérations de filtrage sur un jeu de données à l'aide de la bibliothèque **Pandas** en Python. Il s'agit d'un outil utile pour nettoyer et explorer des données dans le cadre d'une analyse exploratoire.

├── README.md # Fichier de documentation

### 1. Prérequis
